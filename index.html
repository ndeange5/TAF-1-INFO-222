🌐 Qu'est-ce que le web ?

Le web (World Wide Web) est un système hypertexte public fonctionnant sur Internet. Il permet de consulter des pages reliées par des liens hypertextes.

Principes de fonctionnement

  • Client-Serveur : le navigateur (client) envoie des requêtes HTTP à un serveur qui renvoie les ressources.
  • HTTP/HTTPS : protocole de communication entre le client et le serveur.
  • URL : adresse unique pour localiser une ressource.

💻 Langages de base

  • HTML : structure et contenu de la page
  • CSS : mise en forme et style visuel
  • JavaScript : interactivité et dynamisme
  • Frameworks : React, Vue, Angular pour des applications plus complexes
